## 🚀 Quick Start

Simply open `index.html` in your browser:

```bash
# Clone the repository
git clone https://github.com/AliZafar780/alizafar-website.git
cd alizafar-website

# Open in browser
open index.html
# or
xdg-open index.html  # Linux
```
